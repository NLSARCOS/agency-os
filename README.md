# Agency OS

Sistema Operativo total para una agencia de IA.

## Misión
Recibir objetivos de negocio o técnicos y convertirlos en ejecución continua con células expertas: desarrollo, marketing, leadops, ventas, ABM, analítica y creatividad.

## Principios
- autonomía con quality gates
- entregables netos y medibles
- ejecución continua por jobs
- modular, portable e instalable en Linux/Mac

## Capas
1. kernel
2. studios
3. jobs
4. reports
5. install/configs
