# Bootstrap Env

## Objetivo
Dejar Agency OS portable en Linux/Mac.

## Requisitos mínimos
- git
- python3
- nodejs
- npm/pnpm
- gh
- ollama (opcional)
- openclaw

## Estrategia
1. clonar repo/configurar workspace
2. instalar dependencias base
3. copiar env
4. cargar modelos y skills
5. activar cron/jobs
